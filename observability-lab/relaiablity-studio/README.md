# Reliability Studio

**Reliability Studio** is a Grafana App plugin that provides a unified workspace
for SRE & DevOps teams to investigate incidents, track SLOs, and correlate
metrics, logs, traces, and Kubernetes failures.

## Features

- Incident Context Panel  
- SLO & Error Budget Tracking  
- Kubernetes Failure Snapshot  
- Prometheus, Loki, Tempo integration  
- Go backend + React frontend
